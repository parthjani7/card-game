## Steps to setup the project

-   `git clone https://github.com/parthjani7/card-game.git`
-   `rename env.example to .env and add values for database`
-   `php artisan key:generate`
-   `php artisan migrate`
-   `composer install`
-   `npm install`
-   `npm run dev`
-   `php artisan serve`
